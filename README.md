## Getting Started

1. Clone the repository:
  ```bash
  git clone https://github.com/kostyaniekrasov/wt-test-task.git
  ```
2. Install dependencies:
  ```bash
  npm install
  ```

3. Run the development server:
```bash
npm run dev
```
4. Open your browser and navigate to
```bash
http://localhost:3000
```
